[python IDE (통합개발툴)]

1. Anaconda(아나콘다) - 무료(open source) : 
	https://www.anaconda.com/distribution/
	Spyder, IPython, jupyter notebook 등 통합 플랫폼을 제공함.
	프로그램만 한번 설치하면 대부분의 연관된 프로그램을 같이 사용할 수 있음.
	텐서플로우, scipy, pandas 등 요즘 인기있는 대부분의 라이브러리를 제공함.
	데이터 분석을 위한 다양한 라이브러리를 제공함.
	Data Science 분야(딥러닝, 빅데이터 등) 파이썬 개발자들에게 인기가 좋음. 
2. PyCham : 
	https://www.jetbrains.com/ko-kr/pycharm/download/#section=windows
	Jetbrains이란 회사에서 개발한 파이썬 개발 툴임.
	파이썬 개발자들의 생산성, 작업성에 특화된 툴임.
	Professional 버전과 Community 버전 제공, Community 버전은 무료임.
	Community 버전은 일부 기능 제한됨. 무겁다는 단점 있음.
	파이썬 개발자가 가장 많이 사용하는 개발툴임. 
3. Jupyter Notebook :  
	Jupyter notebook 자체 확장자인 .jpynb를 사용하고 있어서 공유가 필요할 때 따로 변환을 해주어야 함.
4. Atom : 
	Python 패키지를 필요시 직접 설치 필요함. 여러 언어 개발 지원
5. Visual Studio Code(VS Code) : 
	MS 사가 제공하는 오픈소스 개발툴. 여러 언어 개발 지원. PyCham 보다 가벼움.
	Python 패키지를 필요시 직접 설치 필요함.
6. Python IDLE : 
	파이썬을 설치할 때 기본적으로 설치되는 개발도구임. 많은 기능을 제공하지는 않음.
